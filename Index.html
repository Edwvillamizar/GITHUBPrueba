GITHUB

GitHub es una plataforma de alojamiento de repositorios Git que proporciona herramientas para el control de versiones de código fuente, seguimiento de problemas y colaboración en proyectos de software..

GitHub

utiliza el sistema de control de versiones distribuido Git para controlar las diferentes versiones del código fuente. Cada vez que se realiza un cambio en el código fuente, se crea una nueva versión que se almacena en el repositorio local del desarrollador
